# Webサイト運用サポートに関する覚書（案）

この覚書は、制作者（以下「甲」）とクライアント（以下「乙」）との間で、Webサイトの保守・運用に関する約束事をまとめたものです。
難しい契約用語を避け、お互いに気持ちよく長く付き合えるためのルールです。

## 1. サポート内容（甲がやること）
月額費用の中で、甲は以下の作業を行います。

1.  **サイトが消えないように守る**
    *   サーバーやドメイン（住所）の管理・更新手続きを代行します。
2.  **LINE一本で修正対応（月額内で対応）**
    *   「値段を変えたい」「写真を差し替えたい」「お知らせを載せたい」などの軽微な修正。
    *   LINEやメールで依頼を受け、原則**2営業日以内**に対応します。
3.  **季節の演出・デザイン調整**
    *   サイトが古びないよう、甲の判断で季節ごとの演出（桜、紅葉など）や、細かなデザイン調整を行います。
4.  **困ったときの相談窓口**
    *   「SNSと連携したい」「怪しいメールが来た」など、Webに関する相談にのります。

## 2. 別料金になること（甲がやらないこと）
以下の作業は、別途お見積りとなります。

*   **ページの大幅な追加:** 「新しい事業の紹介ページを5ページ作りたい」など。
*   **機能の追加:** 「予約システムを入れたい」「ネットショップ機能を本格的に作りたい」など。
*   **素材の撮影・作成:** プロカメラマンによる撮影や、長文のライティングなど。

## 3. 費用と支払い
*   **月額費用:** **16,500円（税込）**
*   **支払い方法:** 毎月末日までに、指定口座へ振込（またはクレジットカード決済）。

## 4. 解約について
*   **いつでも解約可能**です。
*   やめたい月の**前月末**までにご連絡ください。（例：3月末でやめたい場合は、2月末までに連絡）
*   解約後、ドメインやデータの引き渡しが必要な場合は、手数料（1〜2万円程度）にて対応します。
